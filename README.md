# Java 抽卡模拟器

单文件Java实现的鸣潮风格抽卡模拟器，支持单抽/十连抽、保底机制、星声资源系统、无限星声模式。

## 功能特性

- 单抽 / 十连抽
- 五星硬保底（80抽必出）
- 四星保底（10抽必出）
- 大小保底（UP角色50% + 歪了后下次必出UP）
- 软保底（70抽后概率递增）
- 星声消耗与返还（五星+1000、四星+300、三星+50）
- 无限星声模式（一键切换）
- 抽卡记录（按星级着色，点击查看抽数）
- 深色主题UI

## 技术栈

- Java 8+
- Swing GUI
- 随机数概率算法

## 运行方法

javac GachaSimulator.java
java GachaSimulator
